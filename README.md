# EnglishSimulator

WPF приложение для изучения иностранных языков.

## Описание

Настольное приложение для Windows для изучения иностранных языков при помощи интервального метода.
В нем реализована простая система добавления и редактировани колод с пользовательскими предложениями, содержащими аудио и текстовый формат для лучшего усвоения.

## Особенности

- Реализация UI на XAML
- Entity Framework Core для ORM

## Установка и запуск

Для запуска приложения выполните следующую последовательность действий:

1. Клонируйте репозиторий `https://github.com/Dmitriy650-cyber/english-simulator.git`
2. Восстановите зависимости `dotnet restore`
3. Запустите проект `dotnet run`
